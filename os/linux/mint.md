# 快捷键

```
menu
  -> Preferences
  -> Keyboard
  -> Shortcuts
     : Launchers
        -> Launch terminal : 追加快捷键 Win+T
```

补充说明：虽然没找到类似于windows下面“运行”的窗口，但还是可以先点下 <kbd>Win</kbd>，在直接输入要进行的命令：比如 gedit。
